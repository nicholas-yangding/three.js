[name]

基于压缩形式的数据创建纹理, 打开[link:https://en.wikipedia.org/wiki/DirectDraw_Surface DDS]文件查看示例

与[page:CompressedTextureLoader CompressedTextureLoader]一起使用.

构造器

[name]( [param:Array mipmaps], [param:Number width], [param:Number height], [param:Constant format], [param:Constant type], [param:Constant mapping], [param:Constant wrapS], [param:Constant wrapT], [param:Constant magFilter], [param:Constant minFilter], [param:Number anisotropy] )

[page:Array mipmaps] -- mipmaps数组应该包含具有数据，高度和宽度的对象. mipmaps的类型和格式必须正确
[page:Number width] -- 最大的mipmap的宽度
[page:Number height] -- 最大的mipmap的高度
[page:Constant format] -- mipmaps中使用的格式. 其它可选值参见[page:Textures ST3C Compressed Texture Formats], [page:Textures PVRTC Compressed Texture Formats]以及[page:Textures ETC Compressed Texture Format]
[page:Constant type] -- 默认值是[page:Textures THREE.UnsignedByteType]. 其它可选值参见[page:Textures type constants]
[page:Constant mapping] -- 图像是如何应用于对象的. 是一个[page:Textures THREE.UVMapping]类型的对象 其它可选值参见[page:Textures mapping constants]
[page:Constant wrapS] -- 默认值是[page:Textures THREE.ClampToEdgeWrapping]. 其它可选值参见[page:Textures wrap mode constants]
[page:Constant wrapT] -- 默认值是[page:Textures THREE.ClampToEdgeWrapping]. 其它可选值参见[page:Textures wrap mode constants]
[page:Constant magFilter] -- 当纹理元素覆盖多个像素时如何对纹理进行采样。 默认值是[page:Textures THREE.LinearFilter]. 其它可选值参见[page:Textures magnification filter constants]
[page:Constant minFilter] -- 当一个纹理元素覆盖少于1个像素时如何对纹理进行采样。 默认值是[page:Textures THREE.LinearMipMapLinearFilter]. 其它可选值参见[page:Textures minification filter constants]
[page:Number anisotropy] -- 沿着轴穿过具有最高纹素密度的像素的样本数。 默认值是1. 较高的值比基本的mipmap产生更少的模糊结果，代价是使用更多纹理样本. 使用[page:WebGLrenderer.getMaxAnisotropy renderer.getMaxAnisotropy]()找到GPU的最大可靠各向异性值，该值一般是2的幂

属性

共有属性请参阅[page:Texture Texture]基类.

[property:boolean flipY]

默认是false. 纹理翻转对压缩纹理来说没用.

[property:boolean generateMipmaps]

默认是false.压缩纹理生成不了mipmap
