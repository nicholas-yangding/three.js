[name]

创建一个用作深度纹理的纹理. 需要支持[link:https://www.khronos.org/registry/webgl/extensions/WEBGL_depth_texture/ WEBGL_depth_texture]扩展

示例

[example:webgl_depth_texture depth / texture]

构造器

[name]( [param:Number width], [param:Number height], [param:Constant type], [param:Constant wrapS], [param:Constant wrapT], [param:Constant magFilter], [param:Constant minFilter], [param:Number anisotropy], [param:Constant format] )

[page:Number width] -- 纹理宽度.
[page:Number height] -- 纹理高度.
[page:Constant type] -- 默认值是[page:Textures THREE.UnsignedShortType]. 其它选可选值参见[page:Textures type constants]
[page:Constant mapping] -- 详见 [page:Textures type constants]
[page:Constant wrapS] -- 默认值是[page:Textures THREE.ClampToEdgeWrapping]. 其它可选值参见[page:Textures wrap mode constants]
[page:Constant wrapT] -- 默认值是[page:Textures THREE.ClampToEdgeWrapping]. 其它可选值参见[page:Textures wrap mode constants]
[page:Constant magFilter] -- 当一个纹理元素覆盖多个像素时如何对纹理进行采样。 默认值是[page:Textures THREE.NearestFilter]. 其它可选值参见[page:Textures magnification filter constants]
[page:Constant minFilter] -- 当一个纹理元素覆盖少于1个像素时如何对纹理进行采样。 默认值是[page:Textures THREE.NearestFilter]. 其它可选值参见[page:Textures minification filter constants]
[page:Number anisotropy] -- 沿着轴穿过具有最高纹素密度的像素的样本数。 默认值是1. 较高的值比基本的mipmap产生更少的模糊结果，代价是使用更多纹理样本. 使用[page:WebGLrenderer.getMaxAnisotropy renderer.getMaxAnisotropy]()找到GPU的最大可靠各向异性值，该值一般是2的幂
[page:Constant format] -- 必须是[page:Textures DepthFormat] (默认) 和 [page:Textures DepthStencilFormat]二者之一 详见[page:Textures format constants]

属性

共有属性请参阅[page:Texture Texture]基类 - 以下属性中有一些texture类中也有, 但是具有不同的默认值.

[page:Texture.format .format]

值为[page:Textures DepthFormat] (默认) 或[page:Textures DepthStencilFormat]二者之一. 详见[page:Textures format constants]

[page:Texture.type .type]

默认值是 [page:Textures THREE.UnsignedShortType]. 详见[page:Textures format constants]

[page:Texture.magFilter .magFilter]

当一个纹理元素覆盖多个像素时如何对纹理进行采样。 默认值是[page:Textures THREE.NearestFilter]. 其它可选值参见[page:Textures magnification filter constants]

[page:Texture.minFilter .minFilter]

当一个纹理元素覆盖少于1个像素时如何对纹理进行采样 默认值是[page:Textures THREE.NearestFilter]. 其它可选值参见[page:Textures magnification filter constants]

[page:Texture.flipY .flipY]

深度纹理不需要翻转，因此默认情况下为*false*

[page:Texture.generateMipmaps .generateMipmaps]

深度纹理不使用mipmaps.
