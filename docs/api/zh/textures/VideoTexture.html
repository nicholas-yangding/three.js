[page:Texture] →

[name]

创建用于视频纹理的纹理

它几乎与[page:Texture Texture]积累相同, 除了它会一直将[page:Texture.needsUpdate needsUpdate]设为*true*之外，这样纹理就会随着视频播放刷新。另外， 它也不会自动创建[page:Texture.mipmaps mipmaps]

示例

[example:webgl_materials_video materials / video ]

//assuming you have created a HTML video element with id="video" var video = document.getElementById( 'video' ); var texture = new THREE.VideoTexture( video ); texture.minFilter = THREE.LinearFilter; texture.magFilter = THREE.LinearFilter; texture.format = THREE.RGBFormat;

构造器

[name]( [param:Video video], [param:Constant mapping], [param:Constant wrapS], [param:Constant wrapT], [param:Constant magFilter], [param:Constant minFilter], [param:Constant format], [param:Constant type], [param:Number anisotropy] )

[page:Video video] -- 用作纹理的video元素
[page:Constant mapping] -- 图像如何应用于对象，是一个 [page:Textures THREE.UVMapping]类型的对象. 其它可选值参见[page:Textures mapping constants]
[page:Constant wrapS] -- 默认值是[page:Textures THREE.ClampToEdgeWrapping]. 其它可选值参见[page:Textures wrap mode constants]
[page:Constant wrapT] -- 默认值是[page:Textures THREE.ClampToEdgeWrapping]. 其它可选值参见[page:Textures wrap mode constants]
[page:Constant magFilter] -- 当纹理元素覆盖多个像素时如何对纹理进行采样。 默认值是[page:Textures THREE.LinearFilter]. 其它可选值参见[page:Textures magnification filter constants]
[page:Constant minFilter] -- 当纹理元素覆盖少于1个像素时如何对纹理进行采样。 默认值是[page:Textures THREE.LinearMipMapLinearFilter]。 其它可选值参见[page:Textures minification filter constants].
[page:Constant format] -- 纹理中使用的格式 其它可选值参见[page:Textures format constants]
[page:Constant type] -- 默认值是[page:Textures THREE.UnsignedByteType]. 其它可选值参见[page:Textures type constants]
[page:Number anisotropy] -- 沿着轴穿过具有最高纹素密度的像素的样本数 默认值是1. 较高的值比基本的mipmap产生更少的模糊结果，代价是使用更多纹理样本. 使用[page:WebGLrenderer.getMaxAnisotropy renderer.getMaxAnisotropy]()找到GPU的最大可靠各向异性值，该值一般是2的幂

属性

共有属性请参阅[page:Texture Texture]基类.

[property:boolean needsUpdate]

你不需要手动赋值，因为[page:VideoTexture.update update]方法会控制该属性值.

方法

共有方法请参阅[page:Texture Texture]基类.

[method:null update]()

视频每播一帧，就会自动调用一次该方法并将[property:boolean needsUpdate]设为*true*
